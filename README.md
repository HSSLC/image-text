# image-text
用Python製作的模仿字幕拼接圖片製作程式

使用方式
===
需以命令列執行主程式，參數如下：  
`image-text.py 圖片路徑 字幕路徑 [輸出檔名]`  
例如：  
`image-text.py source.jpg 人民的法槌.txt output.jpg`  
字幕檔案要用`\n`做換行

附贈檔案
===
人民的法槌.txt  
讓你快速的玩出第一張圖
